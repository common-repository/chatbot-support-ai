ChatGPT and OpenAI powered AI support chatbot to support your customers. Automatic woocommerce integration included. Just activate the plugin and get an awesome chatbot which can support your site visitors.

== Description ==

This plugin integrates chatgpt in your site. You will have a support AI bot in your site just like OpenAI ChatGPT once you activate this plugin. 
Automatic woocommerce integration included. Go to Settings->Chatbot Support AI for plugin settings. 

Plugin Demo : [Demo Here](http://shop.mansurahamed.com/)

Official Developer: [Mansur Ahamed](https://www.upwork.com/freelancers/~013259d08861bd5bd8).

== Features ==

* You can choose your OpenAI model such as ChatGPT, GPT-3, GPT-4, and GPT-4 32k models
* Add's a free ChatGPT chatbot in your website by default
* Can provide support for 100+ woocommece products
* Can provide support for custom product
* It's possible to use the chat entire site or just one page
* Enable cookies to support chat restore
* Bot can remember chat history
* Easily customizatable with hooks
* Can restrict the chatbot to subscribers/members only
* Settings to set ChatGPT Max Tokens, Temperature, Top_p, Frequency_penalty, Presence_penalty etc.


== Open AI ==

This plugin utilizes the API from [OpenAI](https://beta.openai.com). It does not collect any information from your OpenAI account. It is important for you to check your usage on the [OpenAI website](https://beta.openai.com/account/usage) for accurate information. Please also review their [Privacy Policy](https://openai.com/privacy/) and [Terms of Service](https://openai.com/terms/) for further information.

== Usage ==

1. If the plugin given AI credit expires-
2. Create an account at OpenAI.
3. Create an API key.
4. Save the API key in Settings->Chatbot Support AI, inside your dashboard
5. It's recommended to keep an eye on [your OpenAI usage](https://beta.openai.com/account/usage)!
